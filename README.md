## Vegan-Restaurant : About the project

This is a React Web Application for a Vegan Restaurant. This application uses React Routing to implement multi page web application.

## The Navigation Bar

The Navigation Bar is available on all the pages so as to make the navigation between pages and also back to home page easy.

## The Website has:

## 1.Home Page

## 2.Categories Page

## 3.Products Page

## 4.Contact Page

--->

## The Home Page

The Home Page has a section with short description of all the categories and links to the respective category: Dishes, Desserts and Drinks.

## The Categories Page

The respective page for each category has cards that are links to the product's details.

## The Products Page

This Page is where you can see the details about each individual product. It has nutrition table and an ingredient list.

## The Contact Page

The Contact Page has address details and a form to book table.
